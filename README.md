# A4toA5

将A4文档打印成A4对折之后A5大小的册子，利用打印机的缩小打印功能，在调整好合适的打印顺序，就可以实现。

打印文档页数必须书4的整数页，不是的话，可以随便找个工具在后面补充空白页，补充好后按照顺序打印即可。
